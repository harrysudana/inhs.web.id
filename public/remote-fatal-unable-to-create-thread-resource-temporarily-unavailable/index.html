Remote Fatal Unable to Create Thread Resource Temporarily Unavailable

Mon, Dec 2, 2019

Permasalahan pada saat git pull dari repository client yang menggunakan shared hosting CPanel dengan status remote: fatal: unable to create thread: Resource temporarily unavailable

Hasil pencarian didapatkan hasil dari stackoverflow dengan menaikkan pack.threads di shared hosting milik client.

git config --global pack.threads "5"

Referensi : https://stackoverflow.com/questions/9905257/git-push-fatal-unable-to-create-thread-resource-temporarily-unavailable
